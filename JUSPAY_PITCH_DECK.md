### Talent Commerce Is Broken At Every Level — Not Just Influencers

#### The Same Pain Exists Across All Talent Types

| Pain Point | Nano Influencer (10K followers) | Celebrity (Ranveer Singh) | Cricketer (KL Rahul) | Event Artist (Local Band) |
|---|---|---|---|---|
| **Discovery** | Brand scrolls Instagram for hours | Brand calls talent agency, waits weeks | Brand approaches BCCI-registered agent | Brand calls 20 event managers |
| **Evaluation** | No verified metrics | No ROI data on past endorsements | No brand-fit scoring | No performance history |
| **Pricing** | Creator guesses, brand negotiates | Agent quotes ₹2-5 Cr, no transparency | Black-box pricing | Varies wildly |
| **Payment** | UPI/bank transfer, 30-90 day delay | Complex multi-party contracts | Advance + milestone + royalty | 50% advance, rest after event |
| **Measurement** | Screenshot of insights | PR clippings, no real metrics | Social media mentions count | Footfall estimation |
| **Tax/Compliance** | GST chaos | TDS 194J, multiple entities | Endorsement tax implications | GST on services |

## SLIDE 5 — How It Works

### Universal Workflow — Adapts to Any Talent Type

```
STEP 1: DISCOVER
  Brand selects talent type → Browse verified profiles with real metrics

STEP 2: AI MATCH
  5-model ML ensemble scores talent against brand needs
  Output: Ranked list with match score (0-100) + detailed analysis

STEP 3: ENGAGE
  Influencers → Direct invitation
  Celebrities → Through registered agent/manager portal
  Athletes → Via sports management agency
  Events → Through booking agent or direct
  + Contract generation with digital signatures

STEP 4: PAY (via Juspay)
  Influencer: Brand pays ₹1L → Creator ₹85K + Platform ₹10K + Tax ₹5K
  Celebrity:  Brand pays ₹1Cr → 3 milestones → Celebrity + Agent + Manager + Platform + Tax
  Athlete:    Brand pays ₹50L/season → Monthly releases + performance bonus
  Event:      Brand pays ₹10L → Artist + Venue + Production + Travel + Platform
  ALL via Juspay Split Settlement + Payouts

STEP 5: MEASURE & REPORT
  AI-generated PDF analysis reports (already built!)
  Campaign performance, ROI calculation, recommendations
```
